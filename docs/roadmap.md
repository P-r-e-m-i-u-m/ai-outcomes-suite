# Roadmap

## Phase 1: Open-Source MVP

- TypeScript packages.
- CLI demos.
- PRDs and architecture.
- Example inputs.
- GitHub templates.
- GenAI Lab with RAG, evals, safety scan, and agent planning.

## Phase 2: GenAI Learning System

- 30 small GenAI lessons.
- Lesson template.
- More reusable prompts.
- More eval fixtures.
- RAG datasets for each product.
- Agent workflow examples.

## Phase 3: Web Apps

- SAT tutor dashboard.
- Compliance dashboard with event table and reports.
- Legal agent upload flow.
- GenAI lab workbench.

## Phase 4: Real AI

- Provider abstraction for OpenAI or other LLMs.
- Prompt templates.
- Evaluations.
- Safety tests.
- Embeddings and vector search.

## Phase 5: Commercial Readiness

- Authentication.
- Payments.
- Hosted reports.
- Data deletion flows.
- Legal and compliance review.
