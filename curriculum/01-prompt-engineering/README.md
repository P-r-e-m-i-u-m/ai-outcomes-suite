# Phase 01: Prompt Engineering

## Goal

Write prompts that produce consistent, testable outputs.

## Build

Inspect `prompts/system-product-builder.md` and improve it.

## Artifact

Reusable prompts live in `outputs/prompts/`.

## Quality Bar

A useful prompt should define:

- role
- task
- input format
- output format
- refusal or safety boundaries
- evaluation criteria
