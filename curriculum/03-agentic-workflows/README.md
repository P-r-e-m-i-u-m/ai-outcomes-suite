# Phase 03: Agentic Workflows

## Goal

Turn vague goals into safe, reviewable plans.

## Build

Use `GenAiLab.createAgentPlan()` to produce a tool-using workflow.

## Artifact

Agent specs live in `outputs/agents/`.

## Quality Bar

Good agents:

- clarify the goal
- retrieve context
- draft output
- evaluate themselves
- escalate risky tasks to a human
