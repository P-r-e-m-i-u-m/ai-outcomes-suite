# PRD: Generative AI Engineering Lab

## Problem

Most GenAI repositories are either broad courses or tiny demos. Builders need a practical bridge: learn the core pattern, run it, test it, and connect it to a product.

## MVP

- RAG retrieval and cited answers.
- Prompt evaluation.
- Safety scanning.
- Agent plan generation.
- Curriculum and reusable artifacts.

## Target Users

- AI builders.
- Students learning GenAI engineering.
- Open-source contributors.
- Startup founders prototyping AI products.

## Why It Belongs Here

The three original products need the same GenAI foundations:

- SAT tutor needs grounded explanations and outcomes.
- Compliance OS needs logs, evals, and audit evidence.
- Legal agent needs safety checks and careful drafting.

## First Paid Version

- Hosted GenAI workbench.
- RAG dataset manager.
- Evaluation dashboard.
- Agent workflow builder.
- Team templates for compliance, education, and legaltech.
