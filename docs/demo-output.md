# Demo Output

This is the expected smoke-test behavior after running:

```bash
npm run build
npm run demo:all
```

## SAT Tutor

- Calculates a readiness score.
- Finds weakest SAT Math skills.
- Produces a seven-day plan.
- Shows an outcome policy.

## AI Compliance OS

- Logs AI events.
- Counts high-risk events.
- Detects missing consent.
- Creates a human oversight queue.
- Generates a model-card-style report.

## Personal Legal Agent

- Detects a rental document.
- Flags refund, termination, and deposit risks.
- Extracts the deadline phrase `within 30 days`.
- Drafts a written action letter.
