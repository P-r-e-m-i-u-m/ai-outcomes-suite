# Generative AI Engineering Curriculum

This curriculum turns the repo from a set of MVPs into a practical GenAI engineering lab.

The philosophy:

- Learn the concept.
- Run code.
- Ship a reusable artifact.
- Connect it to one of the three products.

## Path

| Phase | Focus | Artifact |
| --- | --- | --- |
| 00 | GenAI foundations | mental model and glossary |
| 01 | Prompt engineering | reusable prompt templates |
| 02 | RAG systems | retrieval and cited answers |
| 03 | Agentic workflows | tool-using plans |
| 04 | Evals and safety | tests, checks, and risk scans |
| 05 | Productization | packaging, CI, docs, launch |

## How To Use

```bash
npm install
npm test
npm run demo:genai
```

Then read each phase folder and improve one artifact.
